# Login and Access
Dokumen ini menjelaskan cara pengguna mengakses sistem absensi, termasuk proses login, pemulihan password, dan logout dari sistem.

## Cara Login
Ikuti langkah berikut untuk masuk ke sistem absensi :
1. Buka halaman login sistem absensi
2. Masukkan username dan password yang telah terdaftar
3. Klik tombol **Login**
4. Setelah berhasil masuk, pengguna dapat melakukan absensi dengan memilih tombol **Check In** saat mulai bekerja dan **Check Out** saat selesai bekerja

## Lupa Password
Jika pengguna lupa password akun, lakukan langkah berikut : 
1. Klik fitur **Lupa Password** pada halaman login
2. Masukkan alamat email yang sudah terdaftar
3. Klik tombol **Kirim**
4. Periksa email untuk mendapatkan kode verifikasi atau tautan reset password
5. Buat password baru sesuai instruksi yang diberikan 

## Logout Sistem
Untuk keluar dari sistem, ikuti langkah berikut : 
1. Klik profil akun di pojok kanan atas halaman sistem
2. Pilih menu **Logout**
3. Sistem akan mengakhiri sesi pengguna dan kembali ke halaman login
