# Main Features (Karyawan)
Bagian ini memberikan penjelasan mengenai fungsi fitur utama yang digunakan oleh karyawan pada Sistem Absensi berbasis web. Dokumentasi ini tidak mencakup fitur yang digunakan oleh admin.

## Dashboard
Dashboard merupakan halaman utama yang menampilkan ringkasan informasi kehadiran karyawan secara terpusat dan mudah dipahami.
Informasi yang ditampilkan pada halaman dashboard meliputi:
1. Status kehadiran hari ini
2. Jam masuk terakhir
3. Jam pulang terakhir
4. Ringkasan absensi

## Check In
Fitur Check In digunakan oleh karyawan untuk melakukan absensi saat memulai jam kerja.
Langkah penggunaan fitur check in :
1.Pastikan Anda telah login ke sistem
2. Klik tombol **Check In** pada halaman dashboard
3. Sistem akan menyimpan waktu kehadiran dan menampilkan status **Berhasil**

## Check Out
Fitur Check Out digunakan oleh karyawan untuk melakukan absensi pada saat selesai bekerja.
Langkah penggunaan fitur check out :
1. Pastikan Anda telah login ke sistem
2. Pastikan Anda anda sudah melakukan proses Check In sebelumnya
3. Klik tombol **Check Out**
4. Sistem akan menyimpan waktu pulang dan menampilkan status **Berhasil**

## Riwayat Absensi
Halaman Riwayat Absensi berisi rekapitulasi data kehadiran karyawan yang tercatat oleh system.
Informasi yang dapat dilihat pada halaman ini meliputi :
1. Daftar absensi berdasarkan tanggal
2. Status kehadiran karyawan
3. Jam masuk, jam pulang, keterlambatan, cuti, izin
